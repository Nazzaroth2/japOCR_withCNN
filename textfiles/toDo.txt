idea for finding words:
the moment the nn finds a char and is confident about it, we break the analising process and
start to analize once into horizontal space (if we find another char we can stop there and keep going
horizontally, if no word is found this way we restart doing vertically) and if no char is found
we start analizing into vertical space. then we mark the word we found to the space where we found it
and also mark any space we already analized in the process so we dont reanalize them.
